## 5. 화면 전환 흐름 (User Journey)

```
[/upload]  ← 진입점 (/)
   │
   │ PPT 생성 시작 클릭
   │ (파일 업로드 → Job 생성)
   ▼
[/jobs/:jobId]  ──── 5초 폴링 ────┐
   │ SUCCEEDED                    │ PENDING / RUNNING
   │                              └──────────────────
   │ 다운로드 버튼 클릭
   ▼
[output.pptx 다운로드]

[/jobs/:jobId] ← FAILED
   │
   │ 다시 시도하기 클릭
   ▼
[/upload]

[Header] → 히스토리 클릭
   ▼
[/history]
   │
   │ 보기 클릭
   ▼
[/jobs/:jobId]
```
